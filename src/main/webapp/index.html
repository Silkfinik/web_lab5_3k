Система управления "Telecom App"

Добро пожаловать в систему управления абонентами и услугами. Это приложение разработано в рамках Лабораторной работы №5.

Приложение использует Java Servlets, Thymeleaf и JPA (EclipseLink) для взаимодействия с базой данных MySQL.

Меню:

  • Показать всех абонентов
  • Показать все доступные услуги
  • Показать неоплаченные счета
  • (Админ) Очистить и заполнить базу тестовыми данными
